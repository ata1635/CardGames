---

| Package / Class                 | Responsibility / Aufgabe                                                                 |
| ------------------------------- | ---------------------------------------------------------------------------------------- |
| `com.ata1635.scoundrel.model`   | **Datenmodelle:** Repräsentiert die Spielobjekte, keine Spiellogik                       |
| `Card.java`                     | Repräsentiert eine einzelne Karte (Farbe, Wert)                                          |
| `Deck.java`                     | Hält alle Karten, zieht Karten, verwaltet das Kartendeck                                 |
| `Player.java`                   | Repräsentiert einen Spieler (Handkarten, Name, Punkte)                                   |

| `com.ata1635.scoundrel.service` | **Spiellogik & Funktionen:** Regeln, Mischen, Abläufe                                    |
| `GameEngine.java`               | Steuert den Spielablauf, überprüft Regeln, startet und beendet das Spiel                 |
| `DeckShuffler.java`             | Mischt das Deck, optional verschiedene Mischstrategien                                   |

| `com.ata1635.scoundrel.ui`      | **Benutzeroberfläche:** Kommunikation mit dem Spieler                                    |
| `ConsoleUI.java`                | Konsolenbasierte Eingabe/Ausgabe, zeigt Spielstatus, liest Spieleraktionen               |
| `GUI.java`                      | Optional: grafische Oberfläche (Swing/JavaFX), zeigt Karten und Status                   |

| `Main.java`                     | Einstiegspunkt des Programms (`public static void main`), startet `GameEngine`           |

| `src/main/resources`            | **Ressourcen:** Bilder, Sounds, Konfigurationen                                          |
| `src/test/java/...`             | **Tests:** Unit-Tests für `model`, `service`, und `ui`                                   |
| `.gitignore`                    | Dateien, die nicht ins Repository sollen (z. B. `target/`, lokale Notizen, Binärdateien) |
